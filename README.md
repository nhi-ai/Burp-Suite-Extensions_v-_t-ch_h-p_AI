# 🐰💻 Burp Suite Extensions & AI Integration Handbook

### *Cẩm nang nhỏ xinh cho hành trình pentest to lớn* ✨🔐

---

## 🌸 Đây là gì zậy?

Đây là **cẩm nang toàn tập** hướng dẫn từng-bước-một (step-by-step) cách làm chủ **17 Burp Suite Extension** xịn xò nhất, cộng thêm phần **tích hợp AI/LLM** siêu hot (Cloud lẫn Local) cho anh em pentest 🕵️‍♀️🤖

Không có gì đáng sợ đâu, cứ từ từ đọc, từ từ làm lab, rồi bug sẽ tự lòi ra thôi 🐛➡️🎉

---

## 🗂️ Mục lục dễ thương

| 🧸 Nhóm | 🎯 Chủ đề | 🔧 Extension |
|---|---|---|
| 1️⃣ | Kiểm soát truy cập & Session | Autorize, AutoRepeater, Session Auth Fuzzer |
| 2️⃣ | Quét lỗ hổng & Fuzzing | Active Scan++, Backslash Powered Scanner, Param Miner, Turbo Intruder |
| 3️⃣ | Mã hóa & Xử lý dữ liệu | AES Killer, Hackvertor, JWT Editor |
| 4️⃣ | Khai thác công nghệ cụ thể | J2EEScan, InQL/GraphQL Raider, Upload Scanner |
| 5️⃣ | Phân tích & Giám sát | Logger++, Collaborator Everywhere |
| 6️⃣ | 🤖 AI & LLM Extensions | BurpGPT, AI Co-Pilot (Ollama/MCP) |
| 7️⃣ | ⚙️ Tự chế Extension | Montoya API (Custom Scan Check) |

📄 Xem chi tiết đầy đủ trong file nha!

---

## 🚀 Bắt đầu thế nào?

```bash
# Clone em nó về
git clone https://github.com/nhi-ai/burp-suite-extensions-ai-integration.git

# Mở file cẩm nang lên và bắt đầu cày 🌾
cd burp-suite-extensions-ai-integration
```

Sau đó mở Burp Suite lên, pha 1 ly trà sữa 🧋, và bắt đầu từ Nhóm 1 thôi!
